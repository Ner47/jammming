## Description
A simple web app to create a Spotify playlist:

> 1. In the search bar, type the name of the track;
> 2. Select a track from the results list and click `+`;
> 3. If you selected something unnecessary, click on it `-` you can remove it;
> 4. Give your playlist a name and click send.

## Features

For the application to work you need:

1. clientId;
2. redirectUri;
3. accessToken.
